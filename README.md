# Govent-CMS
Govent Ticket Selling Website Management System. 
Completed in October 2023. Developed in PHP.


## 網站技術<br/>
Frontend: HTML/ CSS/ JavaScript/ Bootstrap<br/>
Backend: PHP <br/>
Database: MySQL <br/>


## 網站功能<br/>
  * 登入頁面會員登入、註冊<br/>
  * 個人資料、密碼修改<br/>
  * 主辦單位管理<br/>
  * 活動管理<br/>
  * 優惠券管理<br/>
  * 票卷管理<br/>
  
 
## 主要技能<br/>
以bootstrap為架構，並使用MySQL作為資料庫，主要用PHP達成會員註冊等CRUD功能。<br/>
